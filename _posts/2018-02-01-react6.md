# NPM/NODE

Node.js 는 자바스크립트를 서버환경에서 실행하는 자바스크립트 런타임이다.

NPM 은 Node.js 로 만들어진 모듈들을 설치 및 버전관리 할 수 있는 프로그램이다.

Node.js 를 설치하면 NPM 도 함께 설치됩니다.

LTS 버전으로 설치해주세요.

다운로드: <a href="https://nodejs.org/en/download/">https://nodejs.org/en/download/</a>
